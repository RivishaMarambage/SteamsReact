# **App Name**: Cafe Latte Loyalty System

## Core Features:

- User Authentication: Secure sign-up and login functionality for both customers and admin staff, with role-based access control.
- Menu Management: Admin staff can add, edit, and remove menu items with price changes.
- Loyalty Points Redemption: Customers can redeem loyalty points for discounts or rewards.
- Customer Profiles: Customers can create and edit their profiles, including email, mobile number, name, and cafe nickname.
- Online Ordering for Pickups: Customers can place online orders for pickup.
- Staff Redemption: Staff and admin can redeem customers points by searching their email or mobile number
- Online Ordering Points: Online ordering adds points
- Loyalty Levels: Loyalty has standard levels start with none after 10 point bronz,silver,gold,platinum

## Style Guidelines:

- Primary color: Rich brown (#A67B5B), reminiscent of coffee beans.
- Background color: Light tan (#F2E8D5), creating a warm, inviting atmosphere.
- Accent color: Burnt orange (#D98E45) for highlights and call-to-action buttons.
- Body font: 'PT Sans', a humanist sans-serif for body text.
- Headline font: 'Playfair', a modern serif with an elegant, fashionable feel. Use 'PT Sans' for body text.
- Simple, clean icons for menu categories, profile actions, and loyalty features.
- Subtle transitions and animations when navigating between sections and updating information.
